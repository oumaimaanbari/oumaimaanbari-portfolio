3 Vrije ruimte

3.1 Where do I want to be in ~2 years time?

Over twee jaar wil ik een goede basis hebben in data science en bioinformatica (R en bash), waarmee ik in staat ben om biologische datasets te analyseren en zo bij te dragen aan onderzoeksprojecten.

3.2 How am I doing now with respect to this goal?

Op dit moment ben heb ik de basis van R en bash redelijk onder de knie en ben ik bezig deze skills verder te ontwikkelen. Ik ben comfortabel met data-manipulatie en een paar statistische analyses. Ik heb nog niet veel projecten uitgevoerd, dus hier zal ik nog aan moeten werken om zo mijn vaardigheden te kunnen toe passen en ze kan uitbreiden zodat ik in de toekomst zonder problemen zelfverzekerd een data analyse kan uitvoeren.

3.3 What would be the next skill to learn?

Ik vind CRISPR Cas interesaant dus het lijkt met leuk te leren hoe ik gebruik kan maken van CRISPR-bibliotheken voor genetische screens, met een focus op bioinformatica-analysemethoden voor sequencingdata. Het gebruiken van CRISPR-bibliotheken voor verschillende toepassingen (zoals kanker, neurodegeneratieve ziekten, metabole ziekten, enzovoort). Zo kan ik met sequencing bijvoorbeeld achterhalen met een survival analyse welke genen een kankercel nodig heeft om te overleven. Ook kan ik bioinformatica-tools gebruiken zoals Gene Ontology (GO) analyses, pathway analyses, survival analyses, en metabolite profiling.

3.4 Planning voor de vaardigheid

3.4.1 Stap 1: Doelen opstellen

Mijn doel is om R en Bash te leren gebruiken in de data analyse met gebruik van CRISPR-bibliotheken voor genetische screens die nodig zijn voor sequencingdata. Ik zal hiervoor eerst moeten leren: - Hoe gebruik je CRISPR libraries - Wat voor sequencing analyses ik kan uitvoeren - R en bash leren gebruiken in een data analyse Ik heb gekozen voor deze tutorial: https://bioconductor.org/packages/devel/bioc/vignettes/CrispRVariants/inst/doc/user_guide.html Het R package CrispRVariants kan gebruikt worden om data te analyseren na gebruik van Crispr Cas om de verschillende variaties ten opzichte van reference genome te visualiseren.
