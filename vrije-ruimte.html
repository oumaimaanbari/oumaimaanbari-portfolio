Single Cell RNA sequencing

Single cell RNA sequencing (scRNA-seq) is een techniek waarmee de genexpressie van individuele cellen wordt gemeten. In tegestelling tot bulk RNA sequencing, waarbij de gemiddelde genexpressie van een grote groep ellen tegelijk wordt bepaald. ScRNA-seq maakt het mogelijk om verschillen tussen afzonderlijke cellen zichtbaar te maken.

Bij scRNA-seq wordt RNA uit cellen geïsoleerd en omgezet naar cDNA. Dit kan worden gesequenced en resulteert in een count matrix waarin rijen genen voorstellen en de kolommen de individuele cellen. Per cel staat het aantal reads of Unique Molecular Identifiers (UMIs) per gen.

Het voordeel van scRNA-seq is bijvoorbeeld dat er verschillende celtypen binnen één weefsel onderschieden kunnen worden en verschillen in genexpressie tussen cellen kunnne geanalyseerd worden. ScRNA-seq wordt daarom ook veel toegepast in kankeronderzoek.

Workflow

In de scRNA-seq analysen worden grote datasets gegenereert, Een standaard workflow is als volgt:

  1. Inlezen raw count matrix
  2. Kwaliteitscontrole (QC) per cel
  3. Normalisatie van genexpressiedata
  4. Dimensie reduction met bijvoorbeeld een PCA of UMAP
  5. Clustering van cellen
  6. Identificatie van markergenen

Veel gebruike R packages voor het standaardiseren van de scRNA-seq analyse zijn Seurat, SingleCellExperiment en scran.

raw_mtx <- readMM('counts_unfiltered/cellranger/matrix.mtx') 

genes <- read.csv('counts_unfiltered/cellranger/genes.tsv', sep = '\t', header = F)

rownames(raw_mtx) <- genes[,1] 

colnames(raw_mtx) <- read.csv('counts_unfiltered/cellranger/barcodes.tsv', sep = '\t', header = F)[,1]

# Droplets
## Kansberekening dat de barcode een cel is
out <- emptyDrops(raw_mtx) 
# Standaard minimum voor de kans filteren
keep <- out$FDR <= 0.05 
# Lege drops verwijderen
keep[is.na(keep)] <- FALSE
filt_mtx <- raw_mtx[,keep] 

# QC
## Aantal counts/UMI per cel
nCount_rna <- colSums(filt_mtx)
## Aantal genen met expressie per cell
nFeature_rna <- colSums(filt_mtx > 0)
qc <- data.frame(
  barcode = colnames(filt_mtx),
  nCount_rna = as.numeric(nCount_rna),
  nFeature_rna = as.numeric(nFeature_rna)
)

summary(qc$nCount_rna)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
##     120    5619    7438    8402    9888   60120
summary(qc$nFeature_rna)
##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
##      13    2088    2538    2645    3148    8095
hist(qc$nCount_rna, breaks = 50,
     main = "UMI counts per cel", xlab = "nCount_rna")

Figure 1: Verdeling van totale UMI counts (nCount_rna) per cel na filtering.

De verdeling laat zien dat de meeste cellen relatief lage tot middelhoge UMI counts hebben, met een kleinere groep hogere counts.

hist(qc$nFeature_rna, breaks = 50,

De meeste cellen vertonen een vergelijkbaar aantal gedetecteerde genen, wat wijst op consistente datakwaliteit na filtering.

# Normalisatie
norm_mtx <- t(t(filt_mtx) / colSums(filt_mtx)) * 1e4
log_mtx <- log1p(norm_mtx)

# Variate van elk gen
gen_var <- apply(log_mtx, 1, var)

var_df <- data.frame(
  gene = names(gen_var),
  variance = gen_var
)

hist(var_df$variance,
     breaks = 100,
     main = "Gen variatie na normalisatie",
     xlab = "Variantie (log genormaliseerde expressie)")

Figure 4: Verdeling van gen variatie na normalisatie. Een subset van genen vertoont hoge variatie tussen cellen en is misschien informatief voor downstream analyses.

De meeste genen vertonen lage variatie tussen cellen, terwijl een kleinere subset sterk variërende genen laat zien. Deze genen vormen de basis voor dimensiereductie met bijvoorbeeld UMAP en clustering in downstream scRNA-seq analyses.



Mendelian Randomisation

Mendelian Randomisation (MR) is een statistische methode die wordt gebruikt om causale verbanden te onderzoeken tussen een blootstelling en een uitkomt aan de hand van genetishe varianten. MR baseert zich op het principe dat genetsiche varianten willekeurig worden verdeeld bij de conceptie. Hierdoor kunnen genetische varianten worden gezien als een natuurlijke randomisatie en is de reden dat MR de ´natuurlijke´ randomized control wordt genoemd.

In meeste onderzoeken is het lastig om causale verbanden aan te tonen, omdat resultaten kunnen worden vertekend door confounding factors. MR is hier de oplossing, omdat genetische varianten: vastliggen vòòr ziekteontwikkeling; niet worden beïnvloed door leefstijl en externe omgevingsfactoren; minder gevoelig zijn voor confounding factors.
